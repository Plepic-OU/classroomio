## Regenerating

```bash
cd .claude/skills/c4-model && pnpm install   # once
npx tsx .claude/skills/c4-model/extract.ts
npx tsx .claude/skills/c4-model/generate-diagrams.ts

# Database schema (requires supabase start):
bash .claude/skills/c4-model/db-schema.sh
```
